# 🧸 Aunt Lucy Website

A themed single-page website inspired by **Aunt Lucy from Paddington**, designed as a warm, nostalgic, and interactive personal-style page.

The project combines storytelling, visual design, and client-side interactivity to create a soft, character-driven web experience that feels playful, cozy, and polished.

---

## ✨ Project Overview

This website presents Aunt Lucy as a full character through a dedicated one-page experience that includes:

- an **About Me** section
- a visual **photo gallery**
- a curated **music playlist**
- an interactive **like and emoji animation**
- a live-style **poll section**
- a **Q&A / Web1-Web2-Web3 explanation area**
- a styled **contact form**

The design language was built around soft pastel tones, warm neutrals, rounded elements, and a calm visual identity inspired by Aunt Lucy’s gentle world.

---

## 🎯 Main Goals

The project was built to:

- create a visually cohesive character-based website
- demonstrate strong control over **HTML, CSS, and JavaScript**
- combine content, styling, and interaction in a single polished page
- design a website with a clear personality and narrative feel
- practice building an engaging user experience without frameworks

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts** (`Quicksand`)
- **YouTube embeds** for playlist integration

---

## 🌸 Main Features

### 🧡 Sticky Navigation Menu
A top sticky menu enables smooth navigation between all sections of the page.

### 🐻 About Section
Introduces Aunt Lucy through character storytelling, visual layout, and themed content blocks.

### 📸 Photo Gallery
A styled image gallery with hover captions that reveal descriptions and memories.

### 🎵 Playlist Section
Embedded YouTube videos styled as a personal playlist that fits the character’s emotional world.

### 👍 Like Interaction
A custom like button triggers floating emoji animations for a playful response effect.

### 📊 Poll Section
Interactive voting cards update percentage bars dynamically using JavaScript.

### 💬 Q&A Section
Structured answers presented in decorative cards, covering topics such as **Web1, Web2, and Web3**.

### 📧 Contact Section
Includes:
- a Gmail shortcut link
- a custom contact form
- emoji picker support
- basic client-side validation
