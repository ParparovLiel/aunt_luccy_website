# Aunt Lucy, Personal Website

A responsive RTL website built with HTML, CSS, and vanilla JavaScript.
The goal was to create a clean, structured multi-section site with a consistent visual style and a few lightweight interactive elements, without using frameworks.

## Demo
https://wed-2023.github.io/211937354/

## What’s inside
- Sticky navigation with anchor sections
- RTL layout across the entire site
- Responsive layout using Flexbox and CSS Grid
- Consistent typography (Google Font: Quicksand)
- Card-based sections and a gallery with hover captions

## Interactivity (vanilla JS)
- Like button that spawns animated emojis (DOM manipulation + CSS keyframes)
- Poll component that calculates percentages and updates progress bars, voting is locked after one selection
- Contact form with required-field validation and an emoji picker for the message input

## Tech
- HTML5
- CSS3 (Flexbox, Grid, transitions, animations)
- JavaScript (event listeners, DOM updates)
- RTL support
- Google Fonts
